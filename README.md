# Marky Mark Landing Page

Static landing page for GitHub Pages.

## Files

- `index.html` — main landing page
- `assets/marky-mark-poster.jpg` — poster image
- `assets/marky-mark-theme.m4a` — embedded theme music
- `assets/marky-mark-screenplay.pdf` — screenplay PDF

## GitHub Pages setup

1. Create a GitHub repository, for example `marky-mark`.
2. Upload everything in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.

Your site will publish at your GitHub Pages URL after GitHub finishes building.
